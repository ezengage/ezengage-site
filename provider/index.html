供应商(Provider)¶

国内各个社交网络(SNS)网站和微博网站都提供了开放API，在某种意义上都是用户身份提供者。 通过用户的授权后，可以通过API获得这些网站的用户的信息。

我们称这些网站为身份供应商(Identity Provider), 简称 供应商 (Provider)。

ezEngage 支持的供应商¶

供应商 类型 支持登录 支持内容推送 配置文档
人人网 SNS 是 是 人人网配置指南
豆瓣网 SNS 是 是 豆瓣配置指南
QQ空间 SNS 是 否 [1] QQ配置指南
新浪微博 微博 是 是 新浪微博配置指南
腾讯微博 微博 是 是 腾讯微博配置指南
网易微博 微博 是 是 网易微博配置指南
搜狐微博 微博 是 是 搜狐微博配置指南

Footnotes

[1]QQ空间目前没有开通API来推送内容到QQ空间。
